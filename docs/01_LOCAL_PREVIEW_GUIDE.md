# 01 ローカル表示確認手順

## 手順

1. ZIPを解凍します。
2. 解凍したフォルダー内の `index.html` をダブルクリックします。
3. Chromeで開きます。
4. ログイン画面が表示されます。
5. 「オーナーデモで確認」または「デモユーザーではじめる」を押します。
6. 初回セットアップ後、管理画面に入ります。

## 画面が白い場合

次を確認してください。

- ZIPを解凍せずに中のHTMLを直接開いていないか
- 古い版の `index.html` を開いていないか
- ブラウザの更新ボタンを押したか
- Chromeのキャッシュが残っていないか

それでも白い場合は、Chromeで右クリック → 検証 → Console を開き、赤いエラーをスクショしてください。

## データ保存について

この版はブラウザのlocalStorageに保存します。
そのため、ブラウザやPCを変えるとデータは共有されません。
本番ではFirebase Auth + Firestoreに変更します。
