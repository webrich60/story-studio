# 00 最初に読む手順書

## 目的

このフォルダーは、WEBRICH Story Studio v1.4 をGitHubにアップし、表示確認し、次フェーズへ進むための一式です。

## まず開くファイル

ローカルで確認する場合は、以下をダブルクリックします。

```text
index.html
```

画面が開いたら、まずは次のどちらかで確認します。

- オーナーデモで確認
- デモユーザーではじめる

## 今回の到達点

この版は「完成品」ではなく、本番化の土台です。

できること：

- 管理画面UI確認
- 人生素材・感情メモ入力
- note/Kindle/画像プロンプトのローカル下書き作成
- オーナー無料利用のUI確認
- GitHub Pagesへの公開準備

まだできないこと：

- 本物のログイン
- 他端末とのデータ共有
- AI APIでの本生成
- 決済
- ユーザーごとの本格クラウド保存

## 作業順

1. ローカルで `index.html` を開く
2. UIを確認する
3. GitHubに `webrich-story-studio` リポジトリを作る
4. ファイル一式をアップロードする
5. GitHub Pagesで公開する
6. 表示確認する
7. 問題なければCodexで「調査だけ」を依頼する
8. 次フェーズでFirebase / AI API / 決済を入れる
